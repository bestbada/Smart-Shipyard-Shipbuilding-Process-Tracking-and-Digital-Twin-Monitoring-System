# 스마트 조선소 공정 및 안전 모니터링 시스템 (디지털 트윈)

# 프로젝트 소개
2026 한이음 ICT 멘토링 (해운물류 트랙) 프로젝트임.

조선소 현장이 너무 넓어서 관리자가 일일이 돌아다니며 확인하기 힘들다는 점에 착안함. 
RC카(UGV)와 고정 웹캠을 이용해 현장 영상을 실시간으로 수집하고, 
AI 비전으로 분석해서 웹상의 3D 가상 조선소(디지털 트윈)에 띄워주는 시스템임. 
관리자가 현장에 가지 않아도 원격으로 바로 상황을 파악하는 것이 목표임.

# 주요 기능
- 안전 관리: 작업자 안전모 미착용, 위험구역 접근, 쓰러짐 감지
- 공정 확인: 선박 블록 조립 진행 상황 시각화
- 품질 및 화재 감시: 용접 불량(스패터)이나 초기 연기, 불꽃 탐지

# 사용 기술
- 하드웨어: RC카, 로지텍 C920 웹캠, DJI Tello
- AI 비전: YOLOv8, Pose Estimation
- 백엔드: FastAPI, MongoDB, FFmpeg
- 프론트엔드: React.js, TypeScript, Three.js
